## Notes

- API request flow and LLM workflow use one authoritative primary event.
- Interaction journey intentionally uses linked primary events keyed by `journey.id`, because conversion and drop-off depend on the presence or absence of later steps.
- The matrix is intentionally narrow. Final V1 release framing remains part of `P10.S5`.
