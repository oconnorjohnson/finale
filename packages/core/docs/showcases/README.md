# Showcase Docs

This directory contains the Phase 10 product-shape evidence for V1. Together, these docs show how the same canonical event model supports API request observability, LLM workflow observability, interaction/journey observability, shared queryability expectations, and the final acceptance framing for release review.

- `api-request-flow.md`
- `convex-extension.md`
- `convex-http.md`
- `llm-workflow.md`
- `interaction-journey.md`
- `queryability-matrix.md`
- `v1-acceptance.md`

These files are the documentation evidence surface for `P10.S1` through `P10.S5`, plus the post-V1 Convex extension work in Phase 11.
